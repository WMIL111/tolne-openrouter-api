# Tolne API Customer Guide

Tolne provides an OpenAI-compatible AI inference API.

## Base URL

For local testing:

```text
http://localhost:3000/v1
```

After production deployment, this will become your Tolne API domain, for example:

```text
https://api.tolne.com/v1
```

## Authentication

Use your Tolne API key:

```text
Authorization: Bearer YOUR_TOLNE_API_KEY
```

Do not share your API key publicly.

## Models

```text
http://localhost:3000/models.html
```

## Chat Completions

Endpoint:

```text
POST /v1/chat/completions
```

Example request:

```bash
curl http://localhost:3000/v1/chat/completions \
  -H "Authorization: Bearer YOUR_TOLNE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Qwen/Qwen2.5-7B-Instruct",
    "messages": [
      {
        "role": "user",
        "content": "Hello, introduce yourself in one sentence."
      }
    ]
  }'
```

## Pricing

Pricing is charged by tokens:

```text
Input tokens  = text you send to the model
Output tokens = text generated by the model
```

Current public prices are available at:

```text
http://localhost:3000/models.html
```

## Balance And Recharge

Your account is prepaid. Each successful API request deducts balance based on token usage.

To recharge, contact Tolne support and provide:

```text
Your customer name
Your Tolne API key prefix
Recharge amount
Payment reference, if any
```

## Common Errors

```text
invalid_api_key
```

Your API key is missing or incorrect.

```text
insufficient_balance
```

Your balance is too low. Please recharge.

```text
customer_disabled
```

Your account is disabled. Contact Tolne support.

```text
rate_limit_exceeded
```

You are sending too many requests. Wait and retry.

```text
daily_token_limit_exceeded
```

You reached your daily token limit.
